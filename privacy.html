User Guide

Last Updated: May 15, 2023

1. Getting Started

Welcome to the Restaurant Billing System User Guide. This guide will help you understand how to use the system effectively for restaurant billing and management operations.

This system is designed to simplify billing, order management, and reporting processes in a restaurant setting.

1.1 System Requirements

To use this system, you need:

  • A modern web browser (Chrome, Firefox, Safari, or Edge)
  • JavaScript enabled
  • Internet connection (for initial loading)

1.2 Login Information

For demonstration purposes, the system comes with pre-configured user accounts:

  • Username: admin, Password: admin123
  • Username: staff, Password: staff123

2. Dashboard Overview

The dashboard provides an overview of the restaurant's performance and quick access to key features:

  • Statistics Cards: Display summary information like total sales, orders, and average order value
  • Quick Action Buttons: Provide shortcuts to frequently used functions
  • Recent Activity: Shows recent orders and transactions

2.1 Navigation

The sidebar menu allows you to navigate between different sections of the application:

  • Dashboard: Overview and statistics
  • Billing: Create new bills and process payments
  • Orders: Manage and track orders
  • Reports: View detailed reports and analytics
  • Reset System: Clear demo data (for testing purposes only)

3. Using the Billing System

3.1 Creating a New Bill

To create a new bill:

  1. Navigate to the Billing section from the sidebar
  2. Browse menu items by category (Starters, Main Course, Desserts, Drinks)
  3. Click the "Add" button on an item to add it to the current bill
  4. Adjust quantities using the + and - buttons in the bill panel
  5. The system automatically calculates subtotal, tax, and final amount

3.2 Processing Payments

To process a payment:

  1. Click the "Pay Now" button at the bottom of the bill panel
  2. Select a payment method (Cash, Card, or UPI)
  3. For Card payments: Enter card details when prompted
  4. For UPI payments: Scan the displayed QR code
  5. For Cash payments: Enter amount received and the system will calculate change
  6. Confirm the payment to complete the transaction

4. Managing Orders

The Orders section allows you to track all orders in the system:

4.1 Viewing Orders

To view and manage orders:

  • Navigate to the Orders section from the sidebar
  • Use filter buttons to sort orders by status (All, Pending, Completed, Cancelled)
  • Click "View Details" on any order card to see complete order information

4.2 Order Actions

From the order details modal, you can:

  • Mark an order as Complete
  • Cancel an order (with reason)
  • Print the order details

5. Generating Reports

The Reports section provides analytics and insights:

5.1 Viewing Reports

To access reports:

  • Navigate to the Reports section from the sidebar
  • Select date range using the filter options
  • Click "Apply" to update the reports

5.2 Available Reports

The system provides several types of reports:

  • Sales Report: Detailed breakdown of sales by date
  • Item Report: Most popular menu items and their sales
  • Payment Methods: Analysis of payment methods used
  • Order Completion: Statistics on order completion rates

5.3 Exporting Reports

To export a report:

  • Navigate to the desired report tab
  • Click the "Export" button in the top-right corner
  • Choose the export format (CSV, PDF) when prompted

6. System Customization

6.1 Theme Settings

The system supports light and dark themes:

  • Click the sun/moon icon in the top-right corner of the sidebar to toggle between themes
  • Your theme preference is saved and will be remembered between sessions

6.2 Resetting the System

For demonstration purposes, you can reset the system data:

  • Navigate to the Reset System page from the sidebar
  • Choose what to reset (All Data, Orders Only, Reports Only)
  • Confirm the action when prompted
  • Note: This is intended for testing only and will clear all demo data

7. Troubleshooting

If you encounter issues with the system:

  • Login Problems: Ensure you're using the correct username and password
  • Data Not Saving: Check that your browser allows local storage and cookies
  • Display Issues: Try refreshing the page or switching themes
  • Complete Reset: Use the Reset System function to start fresh
